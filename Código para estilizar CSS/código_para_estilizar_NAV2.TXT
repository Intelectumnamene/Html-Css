Para estilizar os itens de lista (`<li>`) dentro de uma tag `<nav>`, você pode aplicar diversas propriedades CSS para alterar a aparência, o espaçamento e a disposição dos itens. Aqui estão algumas dicas e exemplos de como fazer isso:

### 1. **Estilo Básico**

Primeiro, vamos começar com um estilo básico para os itens de lista dentro de um menu de navegação.

```css
nav ul {
    list-style-type: none; /* Remove os marcadores da lista */
    padding: 0; /* Remove o padding padrão */
    margin: 0; /* Remove a margem padrão */
}

nav li {
    display: inline-block; /* Exibe os itens em linha */
    margin-right: 20px; /* Espaçamento entre os itens */
}

nav li:last-child {
    margin-right: 0; /* Remove o espaçamento do último item */
}

nav a {
    text-decoration: none; /* Remove o sublinhado dos links */
    color: white; /* Define a cor do texto */
    padding: 10px 15px; /* Adiciona espaçamento interno aos links */
}

nav a:hover {
    background-color: #555; /* Cor de fundo ao passar o mouse */
    border-radius: 4px; /* Bordas arredondadas ao passar o mouse */
}
```

### 2. **Estilos para Navegação Vertical**

Se você estiver criando um menu de navegação vertical, pode usar `flex-direction: column;`:

```css
nav ul {
    display: flex; /* Define a lista como flexível */
    flex-direction: column; /* Alinha os itens verticalmente */
}

nav li {
    margin-bottom: 10px; /* Espaçamento entre os itens */
}
```

### 3. **Estilos Responsivos**

Para garantir que o menu de navegação fique bem em diferentes tamanhos de tela, você pode usar media queries:

```css
@media (max-width: 768px) {
    nav ul {
        flex-direction: column; /* Muda a direção para vertical em telas pequenas */
    }

    nav li {
        margin-bottom: 15px; /* Espaçamento maior entre os itens */
    }

    nav li:last-child {
        margin-bottom: 0; /* Remove o espaçamento do último item */
    }
}
```

### 4. **Adicionando Transições e Efeitos**

Você também pode adicionar transições e efeitos para melhorar a experiência do usuário:

```css
nav a {
    transition: background-color 0.3s ease; /* Suaviza a transição da cor de fundo */
}

nav a:hover {
    background-color: #0077cc; /* Cor ao passar o mouse */
    color: #fff; /* Muda a cor do texto ao passar o mouse */
}
```

### 5. **Exemplo Completo**

Aqui está um exemplo completo que combina tudo isso:

```html
<!DOCTYPE html>
<html lang="pt">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Menu de Navegação</title>
    <style>
        nav {
            background-color: #333; /* Cor de fundo da nav */
        }

        nav ul {
            list-style-type: none; /* Remove os marcadores da lista */
            padding: 0; /* Remove o padding padrão */
            margin: 0; /* Remove a margem padrão */
            display: flex; /* Alinha os itens em linha */
        }

        nav li {
            display: inline-block; /* Exibe os itens em linha */
            margin-right: 20px; /* Espaçamento entre os itens */
        }

        nav li:last-child {
            margin-right: 0; /* Remove o espaçamento do último item */
        }

        nav a {
            text-decoration: none; /* Remove o sublinhado dos links */
            color: white; /* Define a cor do texto */
            padding: 10px 15px; /* Adiciona espaçamento interno aos links */
            transition: background-color 0.3s ease; /* Suaviza a transição da cor de fundo */
        }

        nav a:hover {
            background-color: #555; /* Cor de fundo ao passar o mouse */
            border-radius: 4px; /* Bordas arredondadas ao passar o mouse */
        }

        @media (max-width: 768px) {
            nav ul {
                flex-direction: column; /* Muda a direção para vertical em telas pequenas */
            }

            nav li {
                margin-bottom: 10px; /* Espaçamento entre os itens */
            }

            nav li:last-child {
                margin-bottom: 0; /* Remove o espaçamento do último item */
            }
        }
    </style>
</head>
<body>

<nav>
    <ul>
        <li><a href="#home">Início</a></li>
        <li><a href="#sobre">Sobre</a></li>
        <li><a href="#servicos">Serviços</a></li>
        <li><a href="#contato">Contato</a></li>
    </ul>
</nav>

</body>
</html>
```
