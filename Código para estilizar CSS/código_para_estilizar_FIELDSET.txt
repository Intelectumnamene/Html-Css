   ```css
   fieldset {
       border: none;

   fieldset {

   ```css

   }
   ```

   ```css
   legend {
       font-weight: bold;
       font-size: 1.2em;
       color: #0077cc;
       padding: 0 10px; /* Espaçamento ao redor do texto */
   }
   ```

5. **Aplicar fundo e opacidade**:
   ```css
   fieldset {
       background-color: #f7f7f7; /* Fundo */
       opacity: 0.9; /* Transparência */
   }
   ```

6. **Estilizar com gradiente**:
   ```css
   fieldset {
       background: linear-gradient(135deg, #0077cc, #00cc99); /* Gradiente */
